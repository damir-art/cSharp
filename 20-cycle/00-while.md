# while
Самый простой вид цикла. Перед каждой итерацией проверяет условие. Наибольшая вероятность попасть в бесконечный цикл.

    int i = 0;
    while (i < 10)
    {
        Console.WriteLine(i);
        i++;
    }

    Console.ReadKey();

ЗЫ: изменять счетчик можно не в теле цикла, а в условии: while (`i++` < 10)
