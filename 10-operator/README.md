# Операторы в C#

## Виды операций в C#
* Присваивания: `=`
* Арифметические: `+`, `-`, `*`, `/`, `%`
* Строковые:
* Сравнения: `>`, `<`, `==`, `>=`, `<=`, `!=`
* Логические: `И`, `ИЛИ`, `НЕ`

## Инструкция, операции, операторы, операнды

    int sum = 2 + 3;

* Инструкция: `int sum = 2 + 3;`
* Операции: `2 + 3`, `sum = 5`
* Операторы: `+`, `=`
* Операнды: `sum`, `2`, `3`
